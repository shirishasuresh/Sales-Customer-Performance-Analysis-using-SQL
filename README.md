# Sales & Customer Performance Analysis using SQL

## Business Objective
Analyze sales data to understand revenue trends, customer behavior, and product performance using SQL.

## Dataset
- customers: customer demographics and signup date
- orders: order transactions and payment methods
- order_items: product-level sales details

## Tools Used
- MySQL
- Excel
- GitHub

## Key Analysis Areas
- Revenue and sales trends
- Customer spending behavior
- Repeat vs one-time customers
- Product and category performance
- Advanced SQL ranking and window analysis

## SQL Techniques Used
- INNER and LEFT JOIN
- GROUP BY and HAVING
- CASE WHEN logic
- Subqueries
- Window Functions (RANK, LAG)
- Date-based analysis

## Key Insights
- A small group of customers contributes the majority of revenue.
- Electronics category dominates overall sales.
- Repeat customers show higher average order value.
- Certain products drive a disproportionate share of category revenue.

## Business Recommendations
- Focus retention strategies on high-value customers.
- Promote top-performing products to maximize revenue.
- Encourage repeat purchases through loyalty programs.
